> About The Project

"Help a Paw Save a Life" is a web initiative created to address the growing issue of stray animals in Greece.
Our goal is to inform the public, provide actionable advice on how to help strays, and list verified shelters for donations and adoptions.

> Key Features
* Interactive Map: Find nearby animal shelters in Attica using our built-in map (powered by Leaflet.js).
* Actionable Guides: Step-by-step instructions on what to do if you find an injured or stray animal.
* Resources: Direct links to trustworthy organizations and shelters.
* Responsive Design: Fully optimized for mobile, tablet, and desktop devices using Bootstrap 5.
* Emergency Contact: Easy access to contact information for reporting strays.
